# 🧠 Tesis IA Catastro  
## Sistema Automatizado de Validación Documental Catastral mediante NLP

---

## 📌 Descripción del Proyecto

Este proyecto desarrolla un **sistema automatizado de validación documental catastral** basado en técnicas de Procesamiento de Lenguaje Natural (NLP) y comparación estructurada contra bases de datos oficiales.

El sistema permite:

1. 📂 Procesar masivamente archivos PDF catastrales.
2. 🔍 Extraer información relevante mediante técnicas de NLP.
3. 📊 Estructurar y tabular los datos extraídos.
4. 🗄️ Comparar la información contra bases oficiales.
5. 📑 Generar reportes automáticos de inconsistencias.
6. 📈 (Opcional) Visualizar resultados en un dashboard interactivo.

---

## 🎯 Objetivo General

Diseñar e implementar una arquitectura modular reproducible que permita la validación automatizada de información contenida en documentos catastrales utilizando técnicas de Inteligencia Artificial y Ciencia de Datos.

---

## 🏗️ Arquitectura del Sistema

El sistema está estructurado en cinco capas:

1. **Ingesta Documental**
2. **Extracción y Procesamiento NLP**
3. **Estructuración y Normalización**
4. **Validación contra Base Oficial**
5. **Generación de Reportes y Visualización**
